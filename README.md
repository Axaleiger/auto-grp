# Анализатор факторов обводнённости и компенсации закачки

Веб-сервисы на общих данных месторождения.

## Открыть

`https://axaleiger.github.io/auto-grp/`

1. **Закачка** — компенсация по блокам (коридор 90–120%)
2. **Факторы обводнённости** — карта, прорывы, диагностика %В
3. **Запускные параметры** — дебит ННС/ГС, УЭЦН
4. **Данные** (в каталоге) — загрузка JSON-выгрузки из КЛАД

Локально: `start.bat` или `index.html` через локальный сервер.

## Состав

- `index.html` — каталог сервисов + загрузка JSON
- `compensation_dashboard.html` — закачка / компенсация
- `auto_grp_analyzer.html` — факторы обводнённости
- `startup_wells.html` — запускные параметры
- `auto_grp_data.js` — встроенный пакет данных
- `platform_data.js` — загрузка/хранение JSON КЛАД
- `klad/` — скрипт выгрузки и спецификация данных

## Выгрузка из КЛАД

См. [`klad/README.md`](klad/README.md): заполнить `config.json` с `layer_id`, запустить `export_from_klad.py`, загрузить JSON в каталоге → **Данные**.
